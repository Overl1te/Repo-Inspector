Repo Inspector — open-source инструмент анализа качества GitHub-репозиториев через API.
Используя проект, вы соглашаетесь с условиями ниже.

## 1. Назначение

- Проект предназначен для оценки качества и прозрачности репозиториев.
- Проект не предназначен для несанкционированного доступа к чужим данным.
- Проект не гарантирует абсолютную точность проверки и не заменяет аудит безопасности.

## 2. Допустимое использование

Разрешено:

- анализ собственных и публичных репозиториев;
- исследовательское и образовательное использование;
- автоматизация quality-гейтов в CI/CD.

Запрещено:

- попытки обхода ограничений GitHub API;
- использование для нарушений закона, правил GitHub или прав третьих лиц;
- намеренная загрузка вредоносного контента в проект.

## 3. Токены и доступ

- GitHub токен, переданный пользователем в UI/API, используется только для текущего запроса.
- По дизайну токен не сохраняется в базе, файлах отчетов и логике экспорта.
- Пользователь сам отвечает за права и ротацию токена.

## 4. Данные и приватность

- Репозиторий анализируется через GitHub API без клонирования и без запуска кода.
- Отчет может содержать только данные, доступные через API и вычисленные метрики.
- Пользователь отвечает за законность обработки данных конкретного репозитория.

## 5. Отказ от гарантий

ПО поставляется «как есть», без прямых или подразумеваемых гарантий.
Разработчики не несут ответственности за прямой или косвенный ущерб,
вызванный использованием или невозможностью использования проекта,
в пределах, допустимых применимым правом.

## 6. Изменение условий

Условия могут обновляться по мере развития проекта.
Актуальная версия публикуется в репозитории.

## 7. Принятие условий

Если вы используете Repo Inspector, значит вы принимаете эти условия.
Если не согласны — прекратите использование.
